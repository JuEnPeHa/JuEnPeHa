# Hola, soy Juan Enrique.

Soy desarrollador de software, enfocado en crear herramientas multiplataforma y arquitecturas backend robustas. Disfruto construyendo desde el código hasta la infraestructura física que lo soporta.

## Sobre mí

- Actualmente desarrollando arquitecturas backend en Go y clientes en Flutter, integrando agentes de inteligencia artificial para procesamiento de datos complejos.
- Gestiono mi propia infraestructura de laboratorio (self-hosting), optimizando servidores Linux para despliegues de Docker y modelos de inferencia de IA locales.
- Interesado en el ecosistema descentralizado y la integración de hardware a través de diseño 3D paramétrico, scripting en Python y microcontroladores.
- Fuerte preferencia por lenguajes fuertemente tipados y la creación de código escalable.
